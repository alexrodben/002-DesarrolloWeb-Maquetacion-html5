¿Qué es HTML5?

Esto fue posteado el 15/08/2024

Durante los últimos años, el estándar HTML5 se ha estado afianzando en muchas páginas web. Sin embargo, ¿qué podemos decir con eso? ¿Cuál es la diferencia crucial entre HTML "común", por así decirlo, y este nuevo estándar? Hoy hablaremos sobre esto y responderemos a la pregunta de qué es HTML5.

HTML5 es un lenguaje más semántico. En otras palabras, es un lenguaje que permite a los desarrolladores web describir con mayor precisión el contenido de una página web. Se dota de un significado a las diferentes partes de la página web, de forma que el navegador puede distinguir la parte que forma parte de la cabecera gracias a que hay una etiqueta que dice "cabecera", bueno lo dice en inglés "header". ¿Para qué sirve? Ayuda a los buscadores como Google a entender la página web, algo que se espera sea positivo para el SEO, y también para que los navegadores muestren de forma inteligente diferentes versiones de la página web sin tener que usar plugins para dispositivos específicos.

Introducción a CSS3

Esto fue posteado el 15/08/2024

El nombre "hojas de estilo en cascada" viene del inglés "cascading style sheets", del que toma sus siglas. CSS es un lenguaje para definir la presentación de un documento estructurado escrito en HTML. CSS es el lenguaje que describe cómo se debe mostrar un documento escrito en HTML.

¿Para qué sirve el CSS3?

El CSS sirve para definir la estética de un sitio web en un documento externo y eso mismo permite que, modificando este documento, se pueda cambiar la estética entera de un sitio web. El sitio puede variar totalmente de estética cambiando únicamente el CSS, sin tocar para nada los documentos HTML que lo componen.

¿Qué es y para qué sirve JavaScript?

JavaScript es un lenguaje ligero e interpretado, orientado a objetos con funciones de primera clase, más conocido como el lenguaje de script para páginas web. Sin embargo, también se usa en muchos entornos sin navegador, tales como Node.js.
